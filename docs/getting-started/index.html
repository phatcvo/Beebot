# Getting Started with Beebot

Welcome to Beebot! This section will guide you through the installation process, initial configuration, and your first steps with the autonomous mobile robot system.

## Prerequisites

Before installing Beebot, ensure you have the following prerequisites:

- **Ubuntu 18.04 LTS** (Bionic Beaver)
- **ROS Melodic Morenia** (full desktop installation)
- **Python 2.7 / 3.6+**
- **Git** for version control
- **Catkin tools** for building ROS packages

## What's Included

The Beebot system includes the following major components:

:::{.grid}
:::{.g-col-md-6}
### Core Modules
- **SLAM & Localization** - Advanced MCL with reliability estimation
- **Motion Planning** - A* global and DWA local planners
- **Control Systems** - Remote control and hardware interfaces
- **System Integration** - Launch files and visualization
:::

:::{.g-col-md-6}
### Hardware Support
- **2D LiDAR** integration (YDLiDAR TG15)
- **IMU sensors** (Xsens MTi driver)
- **Arduino controllers** with LCD feedback
- **Xbox gamepad** remote control
:::
:::
