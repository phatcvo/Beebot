🤖 Beebot Documentation

Comprehensive documentation for the Beebot Autonomous Mobile Robot System

Overview

Beebot is a sophisticated autonomous mobile robot system designed for indoor navigation and autonomous operations. This documentation provides comprehensive information about the system’s components, APIs, tutorials, and hardware integration.

Key Features

🎯 Advanced Navigation

  • Monte Carlo Localization (MCL) with expansion resetting
  • A* Global Planning with configurable heuristics
  • Dynamic Window Approach (DWA) for local obstacle avoidance
  • Real-time mapping with ray casting algorithms

🔧 System Integration

  • Multi-sensor fusion (LiDAR, IMU, Odometry)
  • Hardware integration with Arduino controllers
  • Remote control via Xbox gamepad
  • MQTT communication for wireless monitoring

## Overview

Beebot is a sophisticated autonomous mobile robot system designed for indoor navigation and autonomous operations. This documentation provides comprehensive information about the system's components, APIs, tutorials, and hardware integration.

## Key Features

:::{.grid}
:::{.g-col-md-6}
### 🎯 Advanced Navigation
- **Monte Carlo Localization (MCL)** with expansion resetting
- **A* Global Planning** with configurable heuristics  
- **Dynamic Window Approach (DWA)** for local obstacle avoidance
- **Real-time mapping** with ray casting algorithms
:::

:::{.g-col-md-6}
### 🔧 System Integration
- **Multi-sensor fusion** (LiDAR, IMU, Odometry)
- **Hardware integration** with Arduino controllers
- **Remote control** via Xbox gamepad
- **MQTT communication** for wireless monitoring
:::
